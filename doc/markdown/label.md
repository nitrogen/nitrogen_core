

## Label Element - #label {}

  The label element produces an HTML span with a CSS type of 'label'.

### Usage

```erlang
   #label { text="Some text.", html_encode=true }

```

### Attributes

   * `text` (string) - The text to display.

   * `html_encode` (boolean) - Set to true to safely html-encode the text.
