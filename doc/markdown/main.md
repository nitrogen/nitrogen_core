## Main Element - #main {}

  Produces the HTML5 Main element.

### Usage

```erlang
   #main { body=[
      "Here is some main content",
      ...
   ]}.

```

### Attributes

   * `body` (Nitrogen Elements) - The body of the HTML5 Main element.

   * `role` (String) - The role of the HTML5 Main element.
