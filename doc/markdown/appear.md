## Appear Action

  Call JQuery's [fadeIn(speed)](http://docs.jquery.com/Effects/fadeIn) on the target element.

### Usage

```erlang
   wf:wire(myDiv, #appear { speed=500 })

```

### Attributes

   * `speed` (integer) - Speed of the effect, in milliseconds.
