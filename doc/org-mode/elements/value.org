* Value Element - #value {}

The value element produces an HTML span with a CSS type of 'value'.

** Usage

#+BEGIN_SRC erlang
   #value { text="Some text." }
#+END_SRC

** Attributes

   + text - (/string/) :: The text to display.

   + html\_encode - (/boolean/) :: Set to true to safely html-encode the displayed value.
