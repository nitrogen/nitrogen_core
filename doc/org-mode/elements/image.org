* Image Element - #image {}

  The image element produces an HTML image.

** Usage

#+BEGIN_SRC erlang
   #image { image="http://hostname.com/path/to/image.png" }
#+END_SRC

** Attributes

   + image - (/string/) :: Set the src of the image.
