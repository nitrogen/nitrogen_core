* Required Field Validator - #is\_required {}

  Validate that the field contains a value.

** Usage

#+BEGIN_SRC erlang
   wf:wire(ButtonID, TextBoxID, #validate { validators=[
     #is_required { text="Required" }
   ]})
#+END_SRC

** Attributes

   + text - (/string/) :: The text to display if validation fails.
