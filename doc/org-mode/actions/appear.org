* Appear Action

  Call JQuery's [[http://docs.jquery.com/Effects/fadeIn][fadeIn(speed)]] on the target element.

** Usage

#+BEGIN_SRC erlang
   wf:wire(myDiv, #appear { speed=500 })
#+END_SRC

** Attributes

   + speed - (/integer/) :: Speed of the effect, in milliseconds.
