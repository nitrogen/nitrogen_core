* Alert Action - #alert {}

  This action tells Nitrogen to trigger a Javascript alert.

** Usage

#+BEGIN_SRC erlang
   wf:wire(#alert { text="Some alert text." })
#+END_SRC

** Attributes

   + text - (/string/) :: The text of the alert.
